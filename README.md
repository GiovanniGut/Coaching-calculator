# Calcolatore Date Coaching

Strumento semplice per calcolare la data di fine di un percorso di coaching in base a:
- Data di inizio
- Numero di sessioni
- Intervallo in settimane tra le sessioni

## Utilizzo

Apri il file `index.html` nel browser oppure usa la versione online:

**ðŸ”— [https://giovannigut.github.io/coaching-calculator](https://giovannigut.github.io/Coaching-calculator)**

## Caratteristiche

- Calcolo automatico della data di fine
- Visualizzazione della durata totale in settimane e mesi
- Default: 8 sessioni ogni 2 settimane
- Funziona offline
- Nessuna installazione richiesta

## Come funziona

1. Seleziona la data di inizio del percorso
2. Imposta il numero di sessioni
3. Imposta l'intervallo in settimane tra una sessione e l'altra
